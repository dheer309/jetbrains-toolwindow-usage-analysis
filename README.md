# jetbrains-toolwindow-usage-analysis

## How to run

### Prerequisite: Anaconda (for Jupyter Notebook)
Download Anaconda 25.5.1 (https://www.anaconda.com/download/success), for this project the distribution installers were used.

### Folder setup

Create a new folder anywhere, then download this code by clicking the "Code" button on the top, and download as a zip. Unzip the zip file, and place the code folder to where you created the new folder.

### Run the code

After that, run `jupyter notebook` from your terminal, or open the Anaconda Navigator and then click on Jupyter Notebook. Then, navigate to the newly created folder and open the Jupyter Notebook. Then click on run all. You can also view the PDF for code and data analysis.
